# Задача 04: Камера следования за шариком

## Приоритет: Высокий
## Оценка: Средняя
## Зависимости: Задача 02

## Описание
Реализовать камеру, которая автоматически следует за шариком от третьего лица в режиме игры.

## Требования
- Камера должна следовать за шариком плавно
- Камера должна быть направлена на шарик
- Камера должна быть за спиной шарика (вид от третьего лица)
- Плавное перемещение камеры (без рывков)
- Кнопка для переключения между режимом следования и свободной камерой

## Технические детали
- Использовать `useFrame` для обновления позиции камеры каждый кадр
- Вычислять позицию камеры относительно шарика
- Использовать lerp для плавного перемещения
- Добавить состояние для переключения режимов камеры

## Критерии приемки
- [ ] Камера следует за шариком в режиме игры
- [ ] Камера направлена на шарик
- [ ] Перемещение камеры плавное
- [ ] Есть возможность переключения режимов камеры
