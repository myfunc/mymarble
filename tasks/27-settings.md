# Задача 27: Настройки игры

## Приоритет: Низкий
## Оценка: Средняя
## Зависимости: Задача 25, Задача 31 (для настроек звука)

## Описание
Реализовать экран настроек игры с различными опциями.

## Требования
- Графические настройки (качество, разрешение, эффекты)
- Настройки звука:
  - Общая громкость
  - Громкость эффектов
  - Включение/выключение звуков
- Управление (настройка клавиш)
- Сохранение настроек в localStorage
- Загрузка настроек при старте игры

## Критерии приемки
- [ ] Экран настроек отображается
- [ ] Можно изменить графические настройки
- [ ] Можно настроить звук (громкость, включение/выключение)
- [ ] Можно настроить клавиши управления
- [ ] Настройки сохраняются в localStorage
- [ ] Настройки загружаются при старте игры
