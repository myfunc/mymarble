# Задача 10: Редактор уровней - базовый UI

## Приоритет: Высокий
## Оценка: Высокая
## Зависимости: Нет

## Описание
Создать базовый интерфейс редактора уровней в стиле Trackmania - минималистичный и компактный.

## Требования
- Минималистичный и компактный UI
- Темная цветовая схема (как в Trackmania)
- Основное пространство отдано 3D сцене
- Панели можно скрывать/показывать
- Игривый, прикольный UI (не скучный)
- Информационная панель (режим, выбранный блок, позиция курсора)
- Панель управления (Сохранить, Загрузить, Тест, Настройки)

## Технические детали
- Создать компоненты панелей
- Использовать CSS для стилизации
- Реализовать переключение режимов редактора
- Добавить состояние для управления UI

## Критерии приемки
- [ ] UI минималистичный и компактный
- [ ] Темная цветовая схема
- [ ] Панели можно скрывать/показывать
- [ ] Информационная панель отображает нужную информацию
- [ ] Панель управления с кнопками работает
