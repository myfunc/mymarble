# Задача 20: Редактор - категории блоков и панель выбора

## Приоритет: Высокий
## Оценка: Высокая
## Зависимости: Задача 10, Задача 06 (система блоков), Задача 32 (для декораций)

## Описание
Реализовать систему категорий блоков с панелью выбора в стиле Trackmania.

## Требования
- Компактная боковая или нижняя панель с категориями блоков
- Каждая категория содержит группу похожих блоков
- Пагинированные иконки объектов внутри каждой категории
- Каждая иконка показывает 3D превью объекта (миниатюра)
- Компактный UI - иконки должны быть компактными
- Система расширяемая для добавления новых объектов и категорий

## Категории блоков:
1. Базовые блоки - кубы, платформы, полублоки
2. Наклонные поверхности - рампы, склоны, полусклоны, угловые склоны
3. Повороты - угловые блоки
4. Платформы - подвижные платформы
5. Препятствия - блоки-убийцы, барьеры, заборы на краю
6. Декорации - визуальные элементы (без физики)
7. Рельсы - система рельсов (см. Задача 33)

## Критерии приемки
- [ ] Панель категорий отображается
- [ ] Можно переключаться между категориями
- [ ] Иконки с 3D превью отображаются
- [ ] Пагинация работает
- [ ] Выбор блока работает
