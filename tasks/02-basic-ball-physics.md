# Задача 02: Базовый шарик с физикой

## Приоритет: Высокий
## Оценка: Средняя
## Зависимости: Задача 01

## Описание
Создать 3D шарик с физическим телом, который подчиняется законам физики (гравитация, столкновения).

## Требования
- Создать 3D модель шарика (сфера)
- Добавить физическое тело (rigid body) для шарика
- Настроить физические параметры (масса, трение, упругость)
- Шарик должен падать под действием гравитации
- Шарик должен корректно сталкиваться с поверхностями

## Критерии приемки
- [x] Шарик отображается в 3D сцене (`Ball` в `GameCanvas`)
- [x] Шарик падает под действием гравитации (динамическое тело Rapier)
- [x] Шарик корректно сталкивается с поверхностями (BallCollider + статичные блоки)
- [x] Физика работает плавно без глитчей (фиксированный шаг, демпфирование)

## Статус
Файл `src/game/components/Ball.jsx` содержит визуализацию и физическое тело с вынесенными параметрами (`BALL_TUNING`). Коллайдер задан явно через `BallCollider`, все материалы получают корректное трение/упругость, что обеспечивает предсказуемые столкновения.
