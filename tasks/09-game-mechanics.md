# Задача 09: Игровые механики (смерть, финиш, таймер)

## Приоритет: Высокий
## Оценка: Средняя
## Зависимости: Задача 02, Задача 06, Задача 08

## Описание
Реализовать основные игровые механики: смерть шарика, финиш уровня и таймер прохождения.

## Требования

### Блок-убийца (смерть шарика)
- Визуально блок должен иметь крест
- При касании шарика - шарик разбивается на мелкие части
- Визуальный эффект разрушения должен быть заметным
- Это считается смертью шарика
- После смерти шарик должен появиться снова в точке спавна

### Финиш и таймер
- Таймер отсчитывает время с начала уровня
- Отображение времени на экране
- Финишная точка (или условие завершения уровня)
- При достижении финиша - остановка таймера
- Отображение результата (время прохождения)

## Критерии приемки
- [ ] Блок-убийца визуально имеет крест
- [ ] При касании шарик разрушается
- [ ] Эффект разрушения заметный
- [ ] Шарик респавнится после смерти
- [ ] Таймер отсчитывает время
- [ ] Время отображается на экране
- [ ] При достижении финиша таймер останавливается
- [ ] Результат отображается
