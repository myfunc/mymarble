# Задача 06: Базовые блоки (кубы)

## Приоритет: Высокий
## Оценка: Низкая
## Зависимости: Задача 01

## Описание
Создать базовые статичные блоки (кубы) для построения уровней.

## Требования
- Создать компонент куба
- Кубы должны быть статичными физическими объектами
- Кубы должны правильно выравниваться по сетке
- Можно ставить куб на куб
- Кубы должны корректно сталкиваться с шариком

## Технические детали
- Использовать `<Box>` из drei или создать mesh
- Создать статичное физическое тело (статический коллайдер)
- Настроить размер куба (например, 1x1x1 единица)
- Обеспечить правильное выравнивание по сетке

## Критерии приемки
- [ ] Кубы отображаются в 3D сцене
- [ ] Кубы статичны и не двигаются
- [ ] Шарик корректно сталкивается с кубами
- [ ] Кубы можно размещать в плотную друг к другу
