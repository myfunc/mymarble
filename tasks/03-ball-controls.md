# Задача 03: Управление шариком

## Приоритет: Высокий
## Оценка: Средняя
## Зависимости: Задача 02

## Описание
Реализовать управление шариком с помощью клавиатуры. Управление должно быть отзывчивым и использовать физические силы.

## Требования
- **КРИТИЧЕСКИ ВАЖНО:** Применять силы, а не менять позицию напрямую
- Стрелки влево/вправо - ускорение влево/вправо
- Стрелка вверх (или пробел) - прыжок (сила вверх)
- Обработка через keydown/keyup события (не текстовый ввод)
- Управление относительно камеры (не глобальное)
- Ускорение должно быть плавным пока кнопка нажата
- Шарик должен быть в 1.5 раза быстрее базовой скорости

## Технические детали
- Использовать `useEffect` с обработчиками `keydown`/`keyup`
- Применять силы через физический движок
- Вычислять направление движения относительно камеры
- Настроить силу ускорения для комфортного геймплея
- Настроить силу прыжка

## Критерии приемки
- [ ] Управление работает резко, без задержек
- [ ] Шарик ускоряется при нажатии клавиш
- [ ] Прыжок применяет силу вверх
- [ ] Управление относительно камеры работает корректно
- [ ] Шарик достаточно быстрый для интересного геймплея
