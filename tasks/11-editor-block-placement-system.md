# Задача 11: Редактор - система размещения блоков

## Приоритет: Высокий
## Оценка: Высокая
## Зависимости: Задача 06, Задача 10

## Описание
Реализовать полную систему размещения блоков в редакторе: размещение, удаление, сетка, выравнивание и поворот.

## Требования

### Размещение блоков
- Режим размещения (по умолчанию)
- Проекция будущей позиции (Ghost Preview) - полупрозрачный блок
- Проекция следует за курсором мыши по сетке
- Клик мыши размещает блок
- Блоки автоматически выравниваются по сетке

### Удаление блоков
- Правая кнопка мыши в режиме размещения удаляет объект под курсором
- Или переключение в режим удаления
- Подсветка объекта перед удалением
- Подтверждение не требуется (как в Trackmania)

### Сетка и выравнивание
- Визуальная сетка на поверхности размещения
- Блоки автоматически выравниваются по сетке
- Можно видеть сетку в 3D пространстве
- Настройка размера сетки (опционально)
- Блоки размещаются в узлах сетки

### Поворот блоков
- Колесико мыши или горячие клавиши для поворота блока перед размещением
- Поворот на 90 градусов (или настраиваемый шаг)
- Визуальная индикация текущей ориентации
- Проекция показывает повернутый блок

## Критерии приемки
- [ ] Ghost-блок отображается перед размещением
- [ ] Ghost-блок следует за курсором
- [ ] Блоки размещаются по клику
- [ ] Блоки выравниваются по сетке
- [ ] Сетка визуально отображается
- [ ] Правая кнопка мыши удаляет блок
- [ ] Объект подсвечивается перед удалением
- [ ] Блоки можно поворачивать перед размещением
- [ ] Поворот работает на 90 градусов
- [ ] Ghost-блок показывает повернутый блок
