# Задача 28: Редактор - инструмент точки спавна

## Приоритет: Средний
## Оценка: Низкая
## Зависимости: Задача 08, Задача 10

## Описание
Реализовать инструмент для выбора и перемещения точки спавна в редакторе.

## Требования
- Специальный инструмент для выбора точки спавна
- Визуально отмечать точку спавна (маркер или блок)
- Можно перемещать точку спавна
- Точка спавна сохраняется в уровне

## Технические детали
- Создать инструмент выбора точки спавна
- Визуальный маркер точки спавна
- Перемещение маркера в редакторе
- Сохранение позиции в данных уровня

## Критерии приемки
- [ ] Можно выбрать инструмент точки спавна
- [ ] Точка спавна визуально отображается
- [ ] Можно переместить точку спавна
- [ ] Позиция сохраняется в уровне
