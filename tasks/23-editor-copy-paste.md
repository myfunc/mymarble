# Задача 23: Редактор - копирование и вставка

## Приоритет: Низкий
## Оценка: Низкая
## Зависимости: Задача 16, Задача 21

## Описание
Реализовать копирование и вставку блоков в редакторе.

## Требования
- Выбрать блок и скопировать (`Ctrl+C`)
- Вставить скопированный блок (`Ctrl+V`)
- Сохраняются все параметры блока (материал, ориентация, настройки)
- Блок вставляется в позицию курсора

## Технические детали
- Хранение скопированного блока в состоянии
- Сериализация параметров блока
- Вставка блока с сохранением всех параметров

## Критерии приемки
- [ ] Можно скопировать блок (Ctrl+C)
- [ ] Можно вставить блок (Ctrl+V)
- [ ] Все параметры блока сохраняются
- [ ] Блок вставляется в правильную позицию
