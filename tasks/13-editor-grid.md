# Задача 13: Редактор - сетка и выравнивание

## Приоритет: Высокий
## Оценка: Средняя
## Зависимости: Задача 11

## Описание
Реализовать визуальную сетку для точного размещения блоков.

## Требования
- Визуальная сетка на поверхности размещения
- Блоки автоматически выравниваются по сетке
- Можно видеть сетку в 3D пространстве
- Настройка размера сетки (опционально)
- Блоки размещаются в узлах сетки

## Технические детали
- Создать компонент сетки (Grid)
- Использовать линии или плоскости для визуализации
- Вычисление ближайшего узла сетки для размещения
- Настройка размера ячейки сетки

## Критерии приемки
- [ ] Сетка визуально отображается
- [ ] Блоки выравниваются по сетке
- [ ] Размер сетки настраивается (опционально)
