# Задача 14: Редактор - поворот блоков

## Приоритет: Средний
## Оценка: Низкая
## Зависимости: Задача 11

## Описание
Реализовать поворот блоков перед размещением.

## Требования
- Колесико мыши или горячие клавиши для поворота блока перед размещением
- Поворот на 90 градусов (или настраиваемый шаг)
- Визуальная индикация текущей ориентации
- Проекция показывает повернутый блок

## Технические детали
- Обработка событий колесика мыши или клавиши R
- Хранение текущей ориентации блока
- Применение поворота к ghost-блоку и размещаемому блоку

## Критерии приемки
- [ ] Блоки можно поворачивать перед размещением
- [ ] Поворот работает на 90 градусов
- [ ] Ghost-блок показывает повернутый блок
