# Задача 16: Редактор - выбор и редактирование объектов

## Приоритет: Средний
## Оценка: Средняя
## Зависимости: Задача 11

## Описание
Реализовать выбор существующих объектов для редактирования их свойств.

## Требования
- Режим выбора объектов
- Левый клик на объект выбирает его
- Выбранный объект подсвечивается
- Панель свойств объекта при выборе
- Изменение материала выбранного объекта
- Для платформ - редактирование точек пути

## Технические детали
- Raycasting для выбора объекта
- Хранение выбранного объекта в состоянии
- Визуальная подсветка выбранного объекта
- Панель свойств с настройками

## Критерии приемки
- [ ] Можно выбрать объект кликом
- [ ] Выбранный объект подсвечивается
- [ ] Панель свойств отображается при выборе
- [ ] Можно изменить свойства объекта
