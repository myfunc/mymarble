# Задача 21: Редактор - горячие клавиши

## Приоритет: Средний
## Оценка: Низкая
## Зависимости: Задача 10, Задача 11, Задача 14

## Описание
Реализовать систему горячих клавиш для быстрого доступа к функциям редактора.

## Требования
- `1-9` - переключение категорий блоков
- `R` - поворот блока
- `Delete` - удаление объекта
- `Ctrl+Z` - отмена действия
- `Ctrl+Y` - повтор действия
- `Ctrl+C` - копировать блок
- `Ctrl+V` - вставить блок
- `T` - тестирование уровня (запуск шарика)
- `Esc` - выход из режима/отмена выбора

## Технические детали
- Создать хук для обработки горячих клавиш
- Обработка комбинаций клавиш (Ctrl+Z и т.д.)
- Интеграция с функциями редактора

## Критерии приемки
- [ ] Все горячие клавиши работают
- [ ] Комбинации клавиш обрабатываются корректно
- [ ] Горячие клавиши не конфликтуют с другими функциями
