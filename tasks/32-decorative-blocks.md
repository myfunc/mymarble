# Задача 32: Декоративные блоки

## Приоритет: Низкий
## Оценка: Средняя
## Зависимости: Задача 06, Задача 20

## Описание
Создать декоративные блоки для украшения уровней. Декоративные блоки не должны влиять на физику, только на визуальное оформление.

## Требования
- Различные декоративные блоки (столбы, колонны, арки, статуи и т.д.)
- Декоративные блоки не имеют физических коллизий
- Декоративные блоки можно размещать в редакторе
- Декоративные блоки поддерживают материалы
- Декоративные блоки можно поворачивать
- Категория "Декорации" в редакторе

## Технические детали
- Создать различные 3D модели декоративных элементов
- Компоненты без физических тел (только визуальные)
- Интеграция с системой категорий блоков
- Поддержка материалов и поворота

## Критерии приемки
- [ ] Декоративные блоки созданы
- [ ] Декоративные блоки не влияют на физику
- [ ] Декоративные блоки можно размещать в редакторе
- [ ] Декоративные блоки поддерживают материалы
- [ ] Декоративные блоки находятся в категории "Декорации"
