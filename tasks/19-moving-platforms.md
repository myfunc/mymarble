# Задача 19: Подвижные платформы

## Приоритет: Средний
## Оценка: Высокая
## Зависимости: Задача 06 (система блоков), Задача 16

## Описание
Создать подвижные платформы, которые движутся по заданному пути.

## Требования
- Платформы движутся по заданному пути
- Можно задать точки пути (waypoints) по которым платформа будет двигаться
- Плавное движение по кругу между точками
- Можно выбрать функцию Безье (Bezier) для перехода между точками
- Можно поставить движение на паузу
- **КРИТИЧЕСКИ ВАЖНО:** Платформа должна выталкивать шарик с инерцией, а не просто перемещать его

## Критерии приемки
- [ ] Платформы движутся по заданному пути
- [ ] Можно задать точки пути
- [ ] Движение плавное
- [ ] Можно использовать кривые Безье
- [ ] Можно поставить на паузу
- [ ] Шарик выталкивается с инерцией, а не телепортируется
