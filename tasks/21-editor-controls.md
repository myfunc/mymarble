# Задача 21: Редактор - управление (горячие клавиши, Undo/Redo, Copy/Paste)

## Приоритет: Средний
## Оценка: Средняя
## Зависимости: Задача 11, Задача 16

## Описание
Реализовать систему управления редактором: горячие клавиши, отмена/повтор действий и копирование/вставка блоков.

## Требования

### Горячие клавиши
- `1-9` - переключение категорий блоков
- `R` - поворот блока
- `Delete` - удаление объекта
- `Ctrl+Z` - отмена действия
- `Ctrl+Y` - повтор действия
- `Ctrl+C` - копировать блок
- `Ctrl+V` - вставить блок
- `T` - тестирование уровня (запуск шарика)
- `Esc` - выход из режима/отмена выбора

### Отмена/Повтор действий (Undo/Redo)
- История действий для отмены/повтора
- `Ctrl+Z` - отменить последнее действие
- `Ctrl+Y` - повторить отмененное действие
- История должна хранить размещение, удаление, изменение объектов
- Ограничение размера истории (опционально)

### Копирование и вставка
- Выбрать блок и скопировать (`Ctrl+C`)
- Вставить скопированный блок (`Ctrl+V`)
- Сохраняются все параметры блока (материал, ориентация, настройки)
- Блок вставляется в позицию курсора

## Критерии приемки
- [ ] Все горячие клавиши работают
- [ ] Комбинации клавиш обрабатываются корректно
- [ ] Горячие клавиши не конфликтуют с другими функциями
- [ ] Можно отменить действие (Ctrl+Z)
- [ ] Можно повторить действие (Ctrl+Y)
- [ ] История хранит все операции
- [ ] Работает корректно для всех типов операций
- [ ] Можно скопировать блок (Ctrl+C)
- [ ] Можно вставить блок (Ctrl+V)
- [ ] Все параметры блока сохраняются
- [ ] Блок вставляется в правильную позицию
