# Задача 33: Система рельсов

## Приоритет: Средний
## Оценка: Высокая
## Зависимости: Задача 06 (система блоков), Задача 19

## Описание
Реализовать систему рельсов (два параллельных стержня) с возможностью создания рельсов по path (пути) с поворотами.

## Требования
- Рельсы состоят из двух параллельных стержней
- Рельсы создаются по path (пути) - задаются точки пути
- Рельсы поддерживают повороты и изгибы
- Шарик может катиться по рельсам
- Рельсы можно редактировать в редакторе (изменять точки пути)
- Визуальное отображение рельсов в 3D
- Плавные переходы между сегментами рельсов

## Технические детали
- Создать систему генерации рельсов по path
- Использовать кривые (splines) для плавных поворотов
- Физические коллизии для рельсов (цилиндры или капсулы)
- Редактор пути для рельсов (аналогично платформам)
- Визуализация рельсов как двух параллельных стержней
- Физика взаимодействия шарика с рельсами

## Критерии приемки
- [ ] Рельсы создаются по path
- [ ] Рельсы поддерживают повороты
- [ ] Шарик может катиться по рельсам
- [ ] Рельсы можно редактировать в редакторе
- [ ] Визуально рельсы выглядят как два параллельных стержня
- [ ] Плавные переходы между сегментами
