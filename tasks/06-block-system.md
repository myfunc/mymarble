# Задача 06: Система блоков (базовые, наклонные, дополнительные)

## Приоритет: Высокий
## Оценка: Высокая
## Зависимости: Задача 01

## Описание
Создать систему блоков для построения уровней: базовые кубы, наклонные блоки (рампы), и дополнительные типы блоков.

## Требования

### Базовые блоки (кубы)
- Создать компонент куба
- Кубы должны быть статичными физическими объектами
- Кубы должны правильно выравниваться по сетке
- Можно ставить куб на куб
- Кубы должны корректно сталкиваться с шариком

### Наклонные блоки (рампы)
- Призматической формы блоки для создания наклонных поверхностей
- Позволяют шарику катиться вниз/вверх
- Шарик должен катиться по рампе, а не улетать вбок
- Физика должна корректно обрабатывать наклонные поверхности

### Дополнительные типы блоков
- **Полублоки** - блоки половинной высоты для создания ступенек и платформ
- **Склоны** - наклонные поверхности (расширение существующих рамп)
- **Полусклоны** - склоны половинной высоты
- **Угловые склоны** - склоны с поворотом для создания угловых переходов
- **Забор на краю** - декоративный/функциональный элемент для краев платформ

### Общие требования для всех блоков
- Все блоки должны корректно работать с физикой
- Все блоки должны поддерживать материалы (см. Задача 18)
- Все блоки должны поддерживать поворот в редакторе

## Критерии приемки
- [ ] Базовые кубы созданы и работают
- [ ] Наклонные блоки (рампы) созданы и работают
- [ ] Полублоки созданы и работают
- [ ] Склоны работают корректно
- [ ] Полусклоны работают корректно
- [ ] Угловые склоны работают корректно
- [ ] Забор на краю создан и работает
- [ ] Все блоки корректно сталкиваются с шариком
- [ ] Шарик корректно катится по наклонным поверхностям
- [ ] Блоки можно размещать в плотную друг к другу
