# Задача 12: Редактор - удаление блоков

## Приоритет: Высокий
## Оценка: Низкая
## Зависимости: Задача 11

## Описание
Реализовать удаление блоков в редакторе.

## Требования
- Правая кнопка мыши в режиме размещения удаляет объект под курсором
- Или переключение в режим удаления
- Подсветка объекта перед удалением
- Подтверждение не требуется (как в Trackmania)

## Технические детали
- Raycasting для определения объекта под курсором
- Обработка правого клика мыши
- Удаление объекта из сцены и данных уровня

## Критерии приемки
- [ ] Правая кнопка мыши удаляет блок
- [ ] Объект подсвечивается перед удалением
- [ ] Удаление работает мгновенно без подтверждения
