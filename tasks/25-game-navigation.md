# Задача 25: Навигация между режимами игры

## Приоритет: Высокий
## Оценка: Низкая
## Зависимости: Задача 10, Задача 24

## Описание
Реализовать навигацию между главным меню, игрой и редактором.

## Требования
- Кнопка "Play" в главном меню открывает режим игры
- Кнопка "Editor" в главном меню открывает редактор
- Кнопка "Settings" в главном меню открывает настройки
- Возврат в главное меню из игры и редактора
- Сохранение состояния при переключении режимов

## Технические детали
- Создать систему управления состоянием приложения
- Реализовать переключение между режимами
- Интеграция с MainMenu компонентом

## Критерии приемки
- [ ] Кнопка "Play" открывает игру
- [ ] Кнопка "Editor" открывает редактор
- [ ] Можно вернуться в главное меню
- [ ] Переключение режимов работает плавно

## Связанные файлы
- `src/App.jsx` - управление режимами
- `src/components/MainMenu.jsx` - интеграция навигации
