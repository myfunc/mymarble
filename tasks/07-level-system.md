# Задача 07: Система уровней (загрузка/сохранение)

## Приоритет: Высокий
## Оценка: Средняя
## Зависимости: Задача 06, Задача 30 (для skybox), Задача 33 (для рельсов)

## Описание
Реализовать систему загрузки и сохранения уровней в формате JSON. Система должна быть расширяемой для поддержки новых типов объектов.

## Требования
- Формат данных для уровня (JSON)
- Функция сохранения уровня в JSON
- Функция загрузки уровня из JSON
- Хранение позиций и типов блоков
- Хранение точки спавна шарика
- Хранение выбранного skybox (см. Задача 30)
- Хранение рельсов и их путей (см. Задача 33)
- Хранение всех параметров блоков (материалы, настройки платформ и т.д.)
- Локальное хранилище (localStorage) для автосохранения
- Система должна быть расширяемой для новых типов объектов

## Технические детали
- Создать структуру данных уровня (расширяемую)
- Реализовать сериализацию/десериализацию
- Использовать localStorage для сохранения
- Создать утилиты для работы с уровнями
- Версионирование формата уровня для обратной совместимости

## Критерии приемки
- [ ] Уровень можно сохранить в JSON
- [ ] Уровень можно загрузить из JSON
- [ ] Все блоки корректно восстанавливаются
- [ ] Точка спавна сохраняется и загружается
- [ ] Skybox сохраняется и загружается (после Задачи 30)
- [ ] Рельсы сохраняются и загружаются (после Задачи 33)
- [ ] Система расширяема для новых типов объектов
