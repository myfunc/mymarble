# Задача 07: Система уровней (загрузка/сохранение)

## Приоритет: Высокий
## Оценка: Средняя
## Зависимости: Задача 06

## Описание
Реализовать систему загрузки и сохранения уровней в формате JSON.

## Требования
- Формат данных для уровня (JSON)
- Функция сохранения уровня в JSON
- Функция загрузки уровня из JSON
- Хранение позиций и типов блоков
- Хранение точки спавна шарика
- Локальное хранилище (localStorage) для автосохранения

## Технические детали
- Создать структуру данных уровня
- Реализовать сериализацию/десериализацию
- Использовать localStorage для сохранения
- Создать утилиты для работы с уровнями

## Критерии приемки
- [ ] Уровень можно сохранить в JSON
- [ ] Уровень можно загрузить из JSON
- [ ] Все блоки корректно восстанавливаются
- [ ] Точка спавна сохраняется и загружается
