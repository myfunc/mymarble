# Задача 17: Наклонные блоки (рампы)

## Приоритет: Средний
## Оценка: Средняя
## Зависимости: Задача 06, Задача 14

## Описание
Создать наклонные блоки (рампы) для создания наклонных поверхностей.

## Требования
- Призматической формы блоки для создания наклонных поверхностей
- Позволяют шарику катиться вниз/вверх
- Можно поворачивать мышкой перед установкой
- Шарик должен катиться по рампе, а не улетать вбок
- Физика должна корректно обрабатывать наклонные поверхности

## Технические детали
- Создать геометрию призмы для рампы
- Добавить физическое тело с наклонной поверхностью
- Настроить трение для корректного качения
- Интегрировать с системой поворота блоков

## Критерии приемки
- [ ] Рампы отображаются в сцене
- [ ] Шарик корректно катится по рампе
- [ ] Рампы можно поворачивать перед размещением
- [ ] Шарик не улетает вбок при качении
