# Задача 29: Дополнительные типы блоков

## Приоритет: Средний
## Оценка: Высокая
## Зависимости: Задача 06, Задача 17

## Описание
Создать дополнительные типы блоков для более разнообразного построения уровней: полублоки, склоны, полусклоны, угловые склоны, забор на краю.

## Требования
- **Полублоки** - блоки половинной высоты для создания ступенек и платформ
- **Склоны** - наклонные поверхности (расширение существующих рамп)
- **Полусклоны** - склоны половинной высоты
- **Угловые склоны** - склоны с поворотом для создания угловых переходов
- **Забор на краю** - декоративный/функциональный элемент для краев платформ
- Все блоки должны корректно работать с физикой
- Все блоки должны поддерживать материалы
- Все блоки должны поддерживать поворот в редакторе

## Технические детали
- Создать геометрию для каждого типа блока
- Добавить физические тела для каждого типа
- Настроить коллизии для корректной работы с шариком
- Интегрировать с системой категорий блоков
- Добавить в редактор с превью

## Критерии приемки
- [ ] Полублоки созданы и работают
- [ ] Склоны работают корректно
- [ ] Полусклоны работают корректно
- [ ] Угловые склоны работают корректно
- [ ] Забор на краю создан и работает
- [ ] Все блоки корректно сталкиваются с шариком
- [ ] Все блоки поддерживают материалы
- [ ] Все блоки можно размещать в редакторе
