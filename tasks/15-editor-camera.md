# Задача 15: Редактор - камера

## Приоритет: Высокий
## Оценка: Высокая
## Зависимости: Задача 10

## Описание
Реализовать свободную камеру в редакторе с несколькими режимами.

## Требования
- Орбитальная камера - вращение вокруг точки
- Свободный полет - WASD + мышь для перемещения
- Следование за курсором - камера следует за курсором мыши
- Горячие клавиши для быстрого переключения режимов камеры
- Сброс позиции камеры к началу уровня

## Технические детали
- Использовать `OrbitControls` из drei для орбитальной камеры
- Реализовать свободную камеру с WASD управлением
- Реализовать режим следования за курсором
- Добавить переключение между режимами

## Критерии приемки
- [ ] Орбитальная камера работает
- [ ] Свободный полет работает (WASD + мышь)
- [ ] Режим следования за курсором работает
- [ ] Можно переключаться между режимами
- [ ] Можно сбросить позицию камеры
