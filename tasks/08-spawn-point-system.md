# Задача 08: Система точки спавна шарика

## Приоритет: Высокий
## Оценка: Низкая
## Зависимости: Задача 02, Задача 07

## Описание
Реализовать систему точки спавна шарика - место, где шарик появляется при старте уровня. Включает базовую функциональность и инструмент для редактора.

## Требования

### Базовая функциональность
- Визуальный маркер точки спавна
- Шарик должен появляться в точке спавна при старте уровня
- Точка спавна должна сохраняться в уровне

### Инструмент в редакторе
- Специальный инструмент для выбора точки спавна
- Можно перемещать точку спавна в редакторе
- Визуально отмечать точку спавна (маркер или блок)
- Позиция сохраняется в данных уровня

## Критерии приемки
- [ ] Точка спавна визуально отображается
- [ ] Шарик появляется в точке спавна при старте
- [ ] Точка спавна сохраняется в уровне
- [ ] Можно выбрать инструмент точки спавна в редакторе
- [ ] Можно переместить точку спавна в редакторе
- [ ] Позиция сохраняется в уровне
