# Задача 26: Таймер и система прохождения уровня

## Приоритет: Средний
## Оценка: Низкая
## Зависимости: Задача 02, Задача 08

## Описание
Реализовать таймер для отслеживания времени прохождения уровня и систему финиша.

## Требования
- Таймер отсчитывает время с начала уровня
- Отображение времени на экране
- Финишная точка (или условие завершения уровня)
- При достижении финиша - остановка таймера
- Отображение результата (время прохождения)

## Технические детали
- Создать компонент таймера
- Отслеживание времени с помощью requestAnimationFrame или Date
- Создать финишную точку/зону
- Обнаружение достижения финиша

## Критерии приемки
- [ ] Таймер отсчитывает время
- [ ] Время отображается на экране
- [ ] При достижении финиша таймер останавливается
- [ ] Результат отображается
