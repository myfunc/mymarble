# Задача 08: Точка спавна шарика

## Приоритет: Высокий
## Оценка: Низкая
## Зависимости: Задача 02, Задача 07

## Описание
Реализовать систему точки спавна шарика - место, где шарик появляется при старте уровня.

## Требования
- Визуальный маркер точки спавна
- Возможность установить точку спавна в редакторе
- Шарик должен появляться в точке спавна при старте уровня
- Точка спавна должна сохраняться в уровне

## Технические детали
- Создать компонент маркера спавна
- Хранить позицию спавна в данных уровня
- При старте уровня телепортировать шарик в точку спавна

## Критерии приемки
- [ ] Точка спавна визуально отображается
- [ ] Шарик появляется в точке спавна при старте
- [ ] Точка спавна сохраняется в уровне
