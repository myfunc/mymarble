# Задача 22: Редактор - отмена/повтор действий (Undo/Redo)

## Приоритет: Средний
## Оценка: Средняя
## Зависимости: Задача 11, Задача 12

## Описание
Реализовать систему истории действий для отмены и повтора операций в редакторе.

## Требования
- История действий для отмены/повтора
- `Ctrl+Z` - отменить последнее действие
- `Ctrl+Y` - повторить отмененное действие
- История должна хранить размещение, удаление, изменение объектов
- Ограничение размера истории (опционально)

## Технические детали
- Создать систему истории действий
- Хранение состояний уровня (snapshots)
- Реализация undo/redo логики
- Интеграция с операциями редактора

## Критерии приемки
- [ ] Можно отменить действие (Ctrl+Z)
- [ ] Можно повторить действие (Ctrl+Y)
- [ ] История хранит все операции
- [ ] Работает корректно для всех типов операций
