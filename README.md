# Ball Game

3D игра про шарик, которым за время надо пройти уровень.

## Технологии

- React 18
- Vite
- React Three Fiber
- Three.js

## Разработка

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр сборки
npm run preview
```

## Деплой

Проект автоматически деплоится на GitHub Pages через GitHub Actions при push в ветку `master` или `main`.
